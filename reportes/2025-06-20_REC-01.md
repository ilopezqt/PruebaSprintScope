## Resumen

Reunión de kick-off con todo el equipo. Se definió el alcance del proyecto, el stack tecnológico y los plazos de entrega.

## Lo que se decidió

- Sistema de recomendación híbrido (colaborativo + content-based)
- Dataset de partida: 500k transacciones de e-commerce del cliente
- Stack: Python (scikit-learn, pandas) + FastAPI + React + PostgreSQL
- Entrega en 6 semanas con hitos intermedios cada 2 semanas
- Métricas de éxito: precision@10 > 0.35, recall@10 > 0.25
