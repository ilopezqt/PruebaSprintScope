# TaskFlow — repo de demo para SprintScope

Repo de prueba para sincronizar con SprintScope.

- `roadmap.md` — roadmap del proyecto (3 fases, jerarquía de subtareas,
  dependencias, áreas, tipos, asignados y descripciones).
- `reportes/` — reportes de sprints ya realizados, en el formato que
  SprintScope parsea (frontmatter YAML + cuerpo markdown).

## Cómo conectarlo a SprintScope

Crea un proyecto nuevo en SprintScope apuntando a este repo (`usuario/TaskFlowDemo`),
o configúralo con las variables `SPRINT_REPOS=usuario/TaskFlowDemo` y
`SPRINT_PROJECT_NAME=TaskFlow`. Personas del equipo: ilopezqt, bgomezqt, tmunozqt.
